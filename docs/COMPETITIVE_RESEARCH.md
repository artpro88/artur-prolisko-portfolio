## INTERACTION PATTERNS

### Scroll Behavior (Lenis.js)
```javascript
// Smooth scroll snap with parallax
- Velocity-based easing
- Section snap (optional)
- Parallax on 3D objects
- Fade effects on sections
```

### 3D Integration (Three.js)
```javascript
// Object behavior tied to scroll
- Scale based on scroll progress (0.8 → 1.2)
- Rotation based on scroll velocity
- Color shift based on section
- Opacity fade for entry/exit
```

### Micro-interactions
- Button hover: Color shift + subtle scale
- Card hover: Lift (transform: translateY) + shadow
- Link hover: Underline animation (width 0 → 100%)
- Section entry: Fade in + slide up (Framer Motion)

- ✅ Server-side rendering for SEO
- ✅ Image optimization
- ✅ Built-in performance features

- ✅ Luxury design system possible with custom config

### Animations & Scroll
**Framer Motion + Lenis.js**
- ✅ React-friendly
- ✅ Scroll-triggered animations
- ✅ Smooth scroll snap
- ✅ 60fps animations

### 3D Graphics
**Three.js + React Three Fiber**
- ✅ WebGL rendering
- ✅ React components for 3D
- ✅ Good performance
- ✅ Plenty of examples

## WHAT NOT TO DO

❌ **Over-animate** - Every motion should serve a purpose  
❌ **Auto-playing audio/video** - Jarring, kills performance  
❌ **Infinite scrolling** - Better: snap sections  
❌ **Heavy hero images** - Optimize aggressively  
❌ **Too many gradients** - Luxury = restraint  
❌ **Comic Sans or ornate fonts** - Use trusted, simple typefaces  
❌ **Bright neons everywhere** - Accent colors only  
❌ **Dated stock photos** - Use real screenshots/professional photography  
❌ **Forgetting mobile** - Assume mobile-first  
❌ **Too much copy** - Edit ruthlessly, value whitespace
